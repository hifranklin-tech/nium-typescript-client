# BankAccountActionState

The state of the action being performed on the bank account.   - `not_started`: The action has not been initiated.   - `in_progress`: The action is in progress.   - `failed`: The action has failed.   - `completed`: The action has completed successfully.

## Enum

* `NotStarted` (value: `'not_started'`)

* `InProgress` (value: `'in_progress'`)

* `Failed` (value: `'failed'`)

* `Completed` (value: `'completed'`)
