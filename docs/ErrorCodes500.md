# ErrorCodes500

The detailed error code associated with HTTP status 500. Avilable values include:   - `fx_dependency_error`: Error happens when the service calls its dependencies.   - `fx_uncategorized_error`: Service errors not categorized.

* `FxDependencyError` (value: `'fx_dependency_error'`)

* `FxUncategorizedError` (value: `'fx_uncategorized_error'`)
