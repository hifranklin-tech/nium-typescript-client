# ExecutionType

Specifies the conversion type. \'Manual\' conversions needs user intervention before \'expiryTime\' to execute, while \'at_conversion_time\' conversions self-execute at \'conversionTime\' without user intervention.

## Enum

* `AtConversionTime` (value: `'at_conversion_time'`)

* `Manual` (value: `'manual'`)
